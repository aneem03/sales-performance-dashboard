# 📊 Sales Performance Dashboard

This project is an interactive Power BI dashboard built using the AdventureWorks dataset.

## 🎯 Objective
Analyze sales performance, product insights, and customer behavior to support business decision-making.

---

## 📁 Pages Overview

### 1️⃣ Executive Overview
- Total Sales, Profit, Profit Margin,Target Achievement 
- Monthly Sales Trend
- Top Products
- Sales by Region

### 2️⃣ Sales Deep Dive
- Sales by Category
- Sales vs Profit Analysis
- Regional performance

### 3️⃣ Customer Insights
- Customer Distribution (Map)
- Sales by Gender
- Sales by Income Level
- Customer KPIs

---

## 🧠 Key Insights
- Bikes generate the highest revenue
- Sales are concentrated in specific regions
- High-income customers contribute most to revenue

---

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling (Star Schema)
